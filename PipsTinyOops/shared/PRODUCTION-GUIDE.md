# PIP COMIC PRODUCTION GUIDE

> Landscape 16:9 · 1920×1080 · 3:00–4:30 · six sequences × 30–45s · 24–30 shots.
> Phase 1 vertical pilot remains a baseline only. New production follows the comic pipeline.

- Channel engine: [`../CHANNEL-STRATEGY.md`](../CHANNEL-STRATEGY.md).

## Production order

1. Lock one want, one escalating problem, one hidden clue and one physical payoff in `VIDEO-PLAN.md`.
2. Write `STORY-SCRIPT.md` before shots. Keep total dialogue/narration at 20 words or fewer.
3. Create `ASSET-MANIFEST.md`; assign exact IDs to Pip, hero set, recurring set angles and story props.
4. Create `SEQUENCE-MAP.md` with six sequence start/end states and one visible story turn each.
5. For each component, write `STORYBOARD.md` first. Every shot specifies asset IDs, opening frame, one action, closing frame, camera and sound job.
6. Generate and approve canonical Pip/set/prop reference images before generating shot keyframes.
7. Generate opening/closing keyframes. Reject identity, prop-count or geography drift before video generation.
8. Compile `PRODUCTION-RUNBOOK.md` from the approved actual frames; prompt describes only the motion between them.
9. Immediately rename every downloaded generation with the episode's deterministic output contract; retain the provider's random source filename in `TAKE-LOG.csv` and never overwrite a prior take.
10. Log the observed endpoint in `TAKE-LOG.csv`; it becomes the next connected shot's opening state.
11. Assemble only approved shots in `SEQUENCE-MAP.md` order and run story-without-sound QA.

## Pip shot rules

- One shot contains one cause, visible consequence and deadpan endpoint.
- Comedy geography must remain readable; prefer locked frames and measured push-ins over decorative camera motion.
- Background details may foreshadow a later solution but cannot move for attention without a physical cause.
- Pip never explains the lesson, addresses the audience or solves the climax with magic.
- Re-anchor Pip identity from the canonical reference after no more than three connected shots.
- After three failed retakes, simplify contact or separate the action; do not add prompt adjectives.
