> **Copy-ready shot guide:** [PRODUCTION-RUNBOOK.md](PRODUCTION-RUNBOOK.md) — full all-in-one prompts, take checklists and one-variable retakes.

> Duration: 32s · output slots: `nouri-v11-c01-s01`…`s04`.
> Job: đặt câu hỏi và cho trẻ dự đoán mà không cue đáp án.

**Set lock:** vertical playroom motion station; mint ramp; ball, cube, book behind identical stoppers; equal light and scale.

| Shot | One action | Required endpoint |
|---|---|---|
| 01 · 8s | Nouri makes one paw sweep, stops, then asks the question | all options still; `WHICH ONE ROLLS?` readable |
| 02 · 8s | visible pointer rail moves attention ball → cube | ball and cube stay still |
| 03 · 8s | pointer rail moves cube → book | all stoppers remain up |
| 04 · 8s | countdown `3 → 2 → 1` | all options still; no test started |

**Handoff:** cut from held three-option frame to the separate cube test set.
**Hard fail:** unequal glow/motion, ball rolling early, extra prop/text, Nouri moving while speaking.
