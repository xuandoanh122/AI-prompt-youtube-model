> **Copy-ready shot guide:** [PRODUCTION-RUNBOOK.md](PRODUCTION-RUNBOOK.md) — full all-in-one prompts, take checklists and one-variable retakes.

> Duration: 32s · output slots: `nouri-v11-c05-s01`…`s04`.
> Job: let the child apply the curved-side rule to a new wheel example.

**Set lock:** mint mat; toy wheel, wooden block, flat card; short ramp and visible release gate.

| Shot | One action | Required endpoint |
|---|---|---|
| 01 · 8s | Nouri asks which new object can roll | three objects equal and still |
| 02 · 8s | true three-second response pause plus countdown | no object moves; digit `1` holds |
| 03 · 8s | gate opens only for wheel; gravity moves it | wheel stops on marked star |
| 04 · 8s | Nouri gives one thumbs-up after speaking | wheel and Nouri hold; no extra text |

**Handoff:** final framing matches opening object scale for a clean platform loop.
**Hard fail:** fake silence, block/card move, gate absent, second gesture or extra speech.
